# Small Animal Motion and Color Flow

Last updated: 2026-06-25

## Purpose

This is the working flow for producing and recoloring DeguDesktop-quality small-animal motion assets in AnimalsDesktop.

It captures the set00 result and the next recommended method for color variants. The repo is self-contained for a Windows clone: the flow packet is generated by the Go command `cmd/flowgridtemplate`, and the QA gates are ordinary Go commands.

The matching reusable Codex Skill on this Mac is installed at:

```text

```

Use `$animal-motion-imagegen-flow` in future Codex threads before starting more motion or color-variant work. On another machine without that Skill, use this document plus `cmd/flowgridtemplate`.

## Provider Decision

| Task | First choice | Reason |
| --- | --- | --- |
| New accepted motion frame | Codex built-in ImageGen, one frame per call | Best retry granularity and accepted-frame provenance. |
| High-risk anatomy repair | Codex built-in ImageGen, one frame per call | Failed frame can be replaced without invalidating the other 61. |
| Color variant pilot | Google Flow | Best candidate for style-preserving image/edit trials if prompt entry is reliable. |
| Prompt critique / alternate wording | ChatGPT or Gemini through sanitized packet | Useful for ideas, not accepted output by itself. |
| Final promotion | Local prepare/audit/assemble/visual QA | Provider output is never accepted without local QA. |

## Color Variant Plan

For color changes, start with an 8-frame pilot:

- `00` idle anchor
- `04` walk start
- `12` fast movement
- `26` species action
- `32` turn
- `40` eat
- `52` groom
- `56` reaction

The target is recoloring only. Preserve:

- silhouette
- pose
- contact baseline
- scale
- eye/muzzle/ear/paw geometry
- tail shape
- motion timing
- transparent or clean chroma background

Hard reject if the output changes species, pose, body shape, baseline, face structure, scale, or adds floor/shadow/text/debris.

## Google Flow Route

Use Google Flow as the first color-variant trial when a visible/manual prompt path is available.

Known local result from 2026-06-25: Flow was reachable and logged in, but automated browser input into the Slate editor did not enable the generation button. For now, treat Flow as manual prompt entry or a browser-control surface that must be revalidated before use.

Recommended first test:

1. Build a small pilot packet under `docs/art-source/external-ai-trials/google-flow-<date>/`.
2. Enter the prompt manually in Flow.
3. Confirm the UI cost/credit state before generation.
4. Generate only the 8-frame pilot.
5. Save raw output under `raw/`.
6. Normalize/crop candidates locally.
7. Review light/dark/checker/contact output.
8. Promote nothing until local QA passes.

Helpful packet command:

```bash
go run ./cmd/flowgridtemplate \
  -out-dir docs/art-source/external-ai-trials/google-flow-20260625/<variant>-color-pilot \
  -animal "<species and target coat>" \
  -sequence set00 \
  -frame-list 0,4,12,26,32,40,52,56 \
  -anchors "docs/art-source/<family>/motion-source/accepted-frames/set00/frame-00.png,docs/art-source/<family>/motion-source/accepted-frames/set00/frame-04.png,docs/art-source/<family>/motion-source/accepted-frames/set00/frame-12.png"
```

This writes the guide images, style anchor sheets, `batch.json`, and `prompt.txt`. The same command works on Windows PowerShell if the line continuations are replaced with backticks or the command is written on one line.

## ChatGPT / InAppBrowser Route

Use ChatGPT/InAppBrowser only with a sanitized packet:

```text
Goal: recolor accepted 96x64 transparent small-animal desktop-pet frames.
Do not change pose, silhouette, baseline, scale, eye/muzzle/ear/paw geometry, tail shape, or motion timing.
Target coat: <coat>.
Reject if there is text, scenery, floor, shadows, duplicate faces, extra animals, cropped anatomy, or style drift.
Return edited images or a concise prompt improvement list.
```

Attach only needed pilot frames or a contact sheet. Do not send raw logs, absolute local paths, credentials, or unrelated repo content.

## Local Acceptance Gates

For each accepted set:

```bash
go run ./cmd/auditframes -frames-dir docs/art-source/<family>/motion-source/accepted-frames/<set> -strict -artifact-warnings

go run ./cmd/assemblemotion \
  -frames-dir docs/art-source/<family>/motion-source/accepted-frames/<set> \
  -out docs/art-source/<family>/motion-source/sheets/<variant>-source-<set>.png \
  -report docs/art-source/<family>/motion-source/accepted-frames/<set>-assemblemotion-report.json

go test -buildvcs=false ./...
go vet -buildvcs=false ./...
git diff --check
```

For release/runtime scope:

```bash
go run ./cmd/importanimals
go run ./cmd/validatemotion -runtime-only -require-accepted
```
