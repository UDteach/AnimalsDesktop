Animals Desktop

チンチラ、ハムスター、マカロニマウス、モモンガ、うさぎが、Windowsのタスクバー上やMacのDock上を歩き回るデスクトップペットです。 Degu Desktopの透明オーバーレイ型デスクトップペットを土台に、複数の小動物へ広げています。

Mac版 Apple Silicon macOS 12+ Mac版 Intel macOS 12+ Windows版は別途ビルド

v0.1.3は5種のset00モーションを入れたプレビューreleaseです。Mac版はZIPを展開して AnimalsDesktop.app を起動してください。 Windows版は別のWindows環境でビルド確認します。

今の版は小動物5種の表示、移動、左右反転、クリック反応、回し車などを確認するためのプレビューです。 10セットすべてを完成させる正式版とは分けて扱っています。

登場する動物

現在アプリ内で選ばれる5種です。各アイコンは実際のランタイムスプライトから作っています。

チンチラ

標準グレー。丸い体、耳、ふさっとした尾をDeguDesktopサイズの動きに収めています。

ハムスター

ゴールデンシリアン。短い尾と丸い体で、回し車にも対応する小動物枠です。

マカロニマウス

太い尾が特徴の小型げっ歯類。低めの姿勢で素早く動く枠です。

モモンガ

膜のシルエットが出るように調整。素材向きの違いもランタイムで補正しています。

うさぎ

チェスナットアグーチ。耳と跳ねる姿勢が読みやすいようにしています。

できること

タスクバー上のペット

透明な最前面レイヤーで、Windowsではタスクバー付近、MacではDock上の下端に小動物がいるように見せます。

最大10匹

設定から表示数を切り替え、複数の小動物を同時に歩かせられます。

自然な左右移動

移動方向とスプライトの向きが合うように描画時に反転し、後ろ向き移動が目立たないようにしています。

クリック反応

動物をクリックすると小さな反応が出ます。デスクトップへの通常クリックは通します。

回し車

小動物向けの回し車表示に対応します。種類ごとの動きに合わせて使うフレームを調整しています。

Mac版

Mac版はメニューバーから設定できます。今回のreleaseからMac ZIPも配布対象に入れています。

これから増やしていく動物

まずベースモーションを作る優先6種です。リリースページでは、次に来るかもしれない動物としてモノクロのシルエットで見せます。

これがくるかも！？

優先対応はファンシーラット、ヤモリ、モルモット、シマリス（アルビノ）、ヨークシャテリア、レオパルド・ゲッコーです。 まず各種のベースモーションを作り、その後に色違い、品種違い、モーフ違いへ広げます。

English

Animals Desktop is a small desktop pet app for Windows and macOS. Chinchilla, hamster, macaroni mouse, sugar glider, and rabbit sprites walk near the Windows taskbar or along the bottom edge above the Mac Dock. Version v0.1.3 is a preview release for the first five set00 motion animals. It is useful for checking the desktop behavior, sprite scale, direction handling, clicks, and wheel behavior before the later full 10-set motion release.
