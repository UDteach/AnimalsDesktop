Animals Desktop

チンチラ、ハムスター、マカロニマウス、モモンガ、うさぎが、Windowsのタスクバー付近や MacのDock上を歩くデスクトップペットです。DeguDesktopの軽量オーバーレイ体験を、複数の小動物へ広げています。

Windows版 Windows 10/11 Mac版 Apple Silicon macOS 12+ Mac版 Intel macOS 12+

v0.1.3は5種のset00モーションを入れたテスト先行公開版です。Windows版はZIPを展開して AnimalsDesktop.exe を起動してください。Mac版はZIPを展開して AnimalsDesktop.app を起動してください。

テスト先行公開版 v0.1.3 Windows版 v0.1.3 Mac版 v0.1.3 動物 5種 フレーム 62 x set00 更新日 2026-06-25
この版は、表示、移動、左右反転、クリック反応、Windows/Mac配布を確認するためのテスト版です。 10セットすべてを完成させる正式ゲートとは分けて扱っています。

登場する動物

現在アプリ内で選べる5種です。未検証の候補種は、公開ページにもアプリにも表示しません。

チンチラ

標準グレー。丸い体、耳、ふさっとした尾をDeguDesktopサイズの動きに収めています。

ハムスター

ゴールデンシリアン。回し車はチンチラとハムスターだけが使います。

マカロニマウス

太い尾が特徴の小動物。低めの姿勢で素早い動きに寄せています。

モモンガ

灰色のシルエットが分かるように調整し、素材差をランタイムで補正しています。

うさぎ

チェスナットアグーチ。耳と跳ねる姿勢が読みやすいサイズにしています。

できること

タスクバー付近のペット

Windowsではタスクバー付近、MacではDock上の下端に小動物を表示します。

最大10匹

設定から表示数を変え、複数の小動物を同時に歩かせられます。

画面設定

Windows版はDeguDesktop由来の表示画面、歩行範囲、縦位置設定を持っています。

クリック反応

動物をクリックすると小さな反応が出ます。デスクトップへの通常クリックは通します。

小物なし

v0.1.3では木、種、餌などの小物を出さず、動物本体の表示と動きだけを確認します。

Windows / Mac配布

v0.1.3ではWindows ZIPとMac ZIPを同じReleaseに置いています。

この版で表示する範囲

このテスト先行公開版では、アプリ内で選べる動物を上記5種に限定しています。 未検証の候補種は、品質が確認できるまで公開ページに並べません。 木、種、餌などの小物表示もこの版では止めています。

English

Animals Desktop is a small desktop pet app for Windows and macOS. Chinchilla, hamster, macaroni mouse, sugar glider, and rabbit sprites walk near the Windows taskbar or along the bottom edge above the Mac Dock. Version v0.1.3 is an early public test release for the first five set00 motion animals. The Windows build now shares the same release tag as the Mac builds. Runtime foraging props are disabled in this preview.
