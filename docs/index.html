Animals Desktop

57種類の小動物が、Windowsのタスクバー付近やMacのDock付近を歩き回るデスクトップペットです。白黒長毛ハムスター2と、赤目で無模様の真のアルビノシマリスも選べます。 作業のそばで控えめに動きながら、小動物たちの歩き、待機、クリック反応を楽しめます。

Windows版 Windows 10/11 Windows版 ネットワーク無効 Windows版 32-bit Mac版 Apple Silicon macOS 12+ Mac版 Intel macOS 12+ SHA256SUMS

v0.2.13 は、57種類の小動物と基本モーションを試せる先行お試し版です。動物選択を安定したID保存に切り替え、過去設定でハシビロコウが別の動物として復元されることがある問題を修正し、動物選択メニューを種類別に整理しました。 Windows版はトレイの Language メニューから表示言語を切り替えられます。Windows版はZIPを展開して AnimalsDesktop.exe を起動してください。Mac版はZIPを展開して AnimalsDesktop.app を起動してください。

ネットワーク無効版は、Smart App Control / SmartScreen などのセキュリティチェックで説明しやすいよう、自動更新確認、トレイからの更新確認、更新ZIPのダウンロードを無効化したWindows 64-bit向け検証版です。Microsoftの説明では、Smart App Controlはクラウド判定が曖昧な場合に有効な署名も確認するため、未署名ビルドの通過を保証するものではありません。詳しくは Smart App Control FAQ と SmartScreen reputation、日本語の補足事例として GlobalSignのSmart App Control事例 を参照してください。

これから追加予定

Pagesに残っていた候補12件は v0.2.9 で追加済みです。次の追加候補は準備中です。

できること

タスクバーやDock付近でお散歩

Windowsではタスクバー付近、MacではDock付近に小動物を表示します。

表示数とサイズ変更

設定から最大10匹までの表示数と、1匹ごとのサイズを変えられます。

表示位置の調整

Windows版では表示する画面、歩き回る範囲、縦位置を調整できます。Mac版では表示先ディスプレイを選べます。

クリック反応

動物をクリックすると小さな反応をします。作業中のクリック操作をできるだけ妨げません。

シンプルな表示

現行プレビューでは木、種、餌などの追加アイテムは表示せず、動物本体のしぐさを中心に楽しめます。

Windows / Mac対応

Windows版 v0.2.13 とMac版 v0.2.13 をこのページから入手できます。

この版について

v0.2.13 は、57種類の小動物を試せるWindows / Macプレビュー版です。 チンチラの色違い2件、ブルーフーディッドのファンシーラット色調補正、白黒長毛ハムスター2、真のアルビノシマリスを含む57種類を選べます。動物選択は安定したIDで保存され、種類別メニューから選びやすくなりました。その他の動物は、動物選択、サイズ変更、表示数、言語、表示先ディスプレイ設定で選べます。 Windows版はトレイから Language を切り替えられます。 木、種、餌などの追加アイテムはこの版では表示しません。
