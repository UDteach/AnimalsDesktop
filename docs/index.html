Animals Desktop

チンチラ、ハムスター、マカロニマウス、モモンガ、うさぎが、Windowsのタスクバー付近や MacのDock付近を歩き回るデスクトップペットです。作業のじゃまをしにくい軽い表示で、 小動物たちの歩き、待機、クリック反応を楽しめます。

Windows版 Windows 10/11 Mac版 Apple Silicon macOS 12+ Mac版 Intel macOS 12+

v0.1.3は、5種類の小動物と基本モーションを試せる先行お試し版です。Windows版はZIPを展開して AnimalsDesktop.exe を起動してください。Mac版はZIPを展開して AnimalsDesktop.app を起動してください。

まずは5種類の小動物と基本的な動きを試せるプレビュー版です。 見た目や動きの品質を確認しながら、公開する動物を順番に増やしていきます。

登場する動物

現在選べる動物は以下の5種類です。品質を確認できた動物から順番に公開しています。

チンチラ

標準グレーのチンチラです。丸い体、大きな耳、ふさっとした尾が分かるように調整しています。

ハムスター

ゴールデンシリアンハムスターです。小さな体でちょこちょこ動く雰囲気を大切にしています。

マカロニマウス

太い尾が特徴の小動物です。低めの姿勢で、すばやく動く様子が分かるようにしています。

モモンガ

灰色の体とシルエットが、小さな表示でも読みやすくなるように調整しています。

うさぎ

チェスナットアグーチのうさぎです。長い耳と、ぴょんと跳ねる姿勢が分かりやすい見た目です。

できること

タスクバーやDock付近でお散歩

Windowsではタスクバー付近、MacではDock付近に小動物を表示します。

最大10匹

設定から表示数を変え、複数の小動物を同時に歩かせられます。

表示位置の調整

Windows版では、表示する画面、歩く範囲、縦位置を好みに合わせて調整できます。

クリック反応

動物をクリックすると小さな反応をします。普段の作業中のクリックは、できるだけ邪魔しないようにしています。

シンプルな表示

v0.1.3では木、種、餌などの追加アイテムを出さず、動物本体の見た目と動きを中心に確認できます。

Windows / Mac配布

Windows版とMac版を同じReleaseページからダウンロードできます。

この版について

v0.1.3は、5種類の小動物を先に試せるお試し版です。 現在は、見た目や動きの品質を確認できた動物だけを掲載しています。 木、種、餌などの追加アイテムはこの版では表示しません。

English

Animals Desktop is a small desktop pet app for Windows and macOS. Chinchilla, hamster, macaroni mouse, sugar glider, and rabbit sprites walk near the Windows taskbar or along the bottom edge above the Mac Dock. Version v0.1.3 is an early public test release for the first five initial-motion animals. The Windows build now shares the same release tag as the Mac builds. Runtime foraging props are disabled in this preview.
