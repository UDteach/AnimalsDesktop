Animals Desktop

Windowsのタスクバー上を、いろいろな小動物が歩き回るデスクトップペットアプリです。 DeguDesktopの軽いWin32オーバーレイ、トレイ操作、キーボード反応、散歩、ホイール、採餌、グルーミングを保ちながら、 100種類の選択可能な動物バリアントへ拡張しています。

Windows版をダウンロード 最新リリース DeguDesktop

画像は現在の100バリアント一覧から作ったプレビューです。順次、ImageGenのsource-truth素材へ置き換えています。

今回の重点

最初のAnimalsDesktopリリースでは、デスクトップペットとして見た目と動きが映える チンチラ、マカロニマウス、ハムスター、ヤモリを優先しています。 既存のデグーに加えて、うさぎ、猫、犬、小動物、爬虫類、両生類まで選べるようになりました。

いま選べる動物

小動物

デグー、チンチラ、マカロニマウス、ハムスター、フェレット、モルモット、ハリネズミ、リス、フクロモモンガ、ラット、マウス、スナネズミなど。

人気の犬・猫

フレンチブルドッグ、ラブラドール、ゴールデンレトリバー、柴犬、ポメラニアン、メインクーン、ラグドール、ペルシャ、ベンガルなど。

うさぎ・野生動物

ホーランドロップ、ネザーランドドワーフ、ミニレッキス、ライオンヘッド、キツネ、レッサーパンダ、カワウソ、カピバラなど。

爬虫類・両生類

ヤモリ、クレステッドゲッコー、フトアゴヒゲトカゲ、リクガメ、コーンスネーク、イエアメガエルなど。

今後の追加予定

  • チンチラ、マカロニマウス、ハムスター、ヤモリのsource-truth素材とモーションを優先して増やします。
  • モモンガ系は、フクロモモンガ、ニホンモモンガ、アメリカモモンガを候補にしています。
  • 小鳥系は、桜文鳥、白文鳥、セキセイインコ、オカメインコ、コザクラインコ、キンカチョウをキューに入れています。
  • 100種類の選択数を保ちつつ、仮素材は順次、種ごとの専用シルエットと生態別モーションへ置き換えます。

English

Animals Desktop is a Windows taskbar pet app based on DeguDesktop. It keeps the lightweight Win32 overlay and tray controls while expanding the selectable catalog to 100 animal variants. Current priorities are chinchillas, macaroni mice, hamsters, geckos, flying squirrels, and small birds.
