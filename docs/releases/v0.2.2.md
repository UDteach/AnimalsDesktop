# AnimalsDesktop v0.2.2

This preview release keeps the current sixteen-animal runtime set and focuses on Mac parity and release-page clarity.

## Downloads

- `AnimalsDesktop-windows-amd64.zip` for Windows 10/11 x64
- `AnimalsDesktop-windows-386.zip` for Windows 10/11 x86
- `AnimalsDesktop-macos-arm64.zip` for Apple Silicon Macs on macOS 12+
- `AnimalsDesktop-macos-amd64.zip` for Intel Macs on macOS 12+
- `SHA256SUMS.txt` for package verification

## What's Changed

- Added Mac animal selection for the same sixteen release-scoped animals as Windows.
- Added Mac per-pet size controls with 70-120% steps.
- Added Mac visible-count, language, and display-selection settings.
- Updated GitHub Pages with JP/EN switching, a compact current-animal grid, Coming Soon silhouettes, and version-change copy.
- Added stricter Mac regression tests for exact animal count, all selectable animals, and size behavior.

## Animal Scope

The selectable animals remain:

- Chinchilla
- Hamster

- Macaroni mouse
- Sugar glider
- Rabbit
- Holland Lop
- Netherland Dwarf
- Himalayan rabbit
- Gecko
- Guinea pig

- Richardson's ground squirrel

Leopard gecko, blue White's tree frog, normal striped chipmunk, birds, cats, and dogs remain planned additions until their runtime assets are promoted.
