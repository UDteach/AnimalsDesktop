# AnimalsDesktop v0.2.7

This is a preview release for 40 selectable animals on Windows and macOS.

## Changes

- Temporarily removes `true_albino_chipmunk` from the public runtime picker and GitHub Pages current-animal list after thumbnail QA found the sprite too weak on white backgrounds and the red eye read too much like a marker dot.
- Keeps the accepted source evidence cataloged so the no-pattern true albino chipmunk repair lane can continue after this release.
- Keeps the v0.2.6 runtime controls and release surface: Windows and Mac animal selection, per-pet size, visible count, language switching, and display settings.

## Known Preview Scope

The public runtime uses accepted `set00` motion sources duplicated into runtime sets. This remains a preview exception before the final full 10-set motion gate.
