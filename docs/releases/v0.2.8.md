# AnimalsDesktop v0.2.8

This is a preview release for 42 selectable animals on Windows and macOS.

## Changes

- Adds `lionhead_rabbit_brown_white` as a selectable ImageGen-derived accepted `set00` animal.
- Adds `shoebill_stork` as a selectable special low-motion accepted `set00` animal.
- Moves Lionhead rabbit and shoebill from the GitHub Pages Upcoming section into the current animal list.
- Keeps `true_albino_chipmunk` out of the public runtime until a new ImageGen repair or edit lane passes white-background readability review.

## Known Preview Scope

The public runtime uses accepted `set00` motion sources duplicated into runtime sets. This remains a preview exception before the final full 10-set motion gate.
