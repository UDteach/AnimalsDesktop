# AnimalsDesktop v0.2.12

Release date: 2026-07-03

## Changes

- Adds `longhair_hamster_black_white_masked` as a second black-and-white longhair hamster source family based on the user-provided masked longhair direction.
- Restores `true_albino_chipmunk` to the public runtime with a fresh ImageGen-only Direction B source: red eyes, no stripes, no dorsal markings, and cream-white coat.
- Updates the runtime picker, GitHub Pages current-animal grid, public preview assets, and release checks for the 57-animal roster.

## Runtime Scope

- Public selectable animals: 57.
- The legacy black-eyed white chipmunk remains cataloged as source evidence only.
- The new hamster and true albino chipmunk are both accepted `set00` ImageGen-derived motion sources.
