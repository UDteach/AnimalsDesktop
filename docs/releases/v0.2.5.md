# AnimalsDesktop v0.2.5

This is a preview release for 41 selectable animals on Windows and macOS, plus a Windows mixed-DPI multi-monitor hotfix.

## Downloads

- `AnimalsDesktop-windows-amd64.zip` for Windows 10/11 x64
- `AnimalsDesktop-windows-386.zip` for Windows 10/11 x86
- `AnimalsDesktop-macos-arm64.zip` for Apple Silicon Macs on macOS 12+
- `AnimalsDesktop-macos-amd64.zip` for Intel Macs on macOS 12+
- `SHA256SUMS.txt` for package verification

## Added

- Blue-green parrotlet
- True albino chipmunk
- Miniature Schnauzer
- Japanese giant salamander
- White wagtail
- Tabby-white domestic shorthair

## Fixed

- Fixed Windows span/multi-monitor mode when monitors have different resolutions, taskbar bottoms, or DPI scaling.
- Split the Windows overlay into per-monitor transparent strips while preserving one shared horizontal walking scene.
- Made local Windows builds per-monitor DPI aware and scaled each monitor segment from a 96 DPI logical canvas to that monitor's DPI.
- Fixed click, hover-name, wheel, forage, and reaction rendering to use the per-monitor segment offset.

## Improved

- Refreshes selected accepted assets for budgerigar, lovebird, quokka, Roborovski hamster, Ragdoll, Scottish Fold, and Toy Poodle.
- Keeps the existing green parrotlet and adds the blue-green parrotlet as a separate selectable variant.
- Renames the previous `albino_chipmunk` runtime entry to black-eyed white chipmunk, and adds a separate true albino chipmunk.
- Corrects leopard gecko turn-frame sizing in the runtime renderer without replacing the accepted source sheet.

## Validation

- Verified on a three-display Windows setup including a secondary monitor at 150% scaling before this animal-roster merge.
- Local release-prep validation covered import regeneration, Pages verification, motion validation, Go tests, Go vet, macOS ZIP builds, Windows cross-builds, and JP/EN Pages smoke checks.

## Pages Roadmap

The GitHub Pages upcoming section remains explicit about the next lanes: the remaining Pages candidates, the white lionhead-pattern rabbit, and the special low-motion shoebill.
