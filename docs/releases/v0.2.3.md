# AnimalsDesktop v0.2.3

This hotfix keeps the v0.2.2 Windows/Mac preview scope and fixes a Windows settings UI clipping issue.

## Downloads

- `AnimalsDesktop-windows-amd64.zip` for Windows 10/11 x64
- `AnimalsDesktop-windows-386.zip` for Windows 10/11 x86
- `AnimalsDesktop-macos-arm64.zip` for Apple Silicon Macs on macOS 12+
- `AnimalsDesktop-macos-amd64.zip` for Intel Macs on macOS 12+
- `SHA256SUMS.txt` for package verification

## What's Fixed

- Fixed the Windows name-change dialog so the Save and Cancel buttons stay fully visible.
- Added regression tests for the rename dialog client-area layout.
- Added a settings footer layout guard so Language, Reset, and Close remain inside the settings window.

## Notes

- This is still a preview release.
- The selectable animal set remains the same sixteen animals as v0.2.2.
- `v0.2.3` is a UI hotfix; it does not promote new runtime animals.
