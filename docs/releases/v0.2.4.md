# AnimalsDesktop v0.2.4

This preview release expands the selectable runtime roster from sixteen to thirty-five animals on Windows and macOS.

## Downloads

- `AnimalsDesktop-windows-amd64.zip` for Windows 10/11 x64
- `AnimalsDesktop-windows-386.zip` for Windows 10/11 x86
- `AnimalsDesktop-macos-arm64.zip` for Apple Silicon Macs on macOS 12+
- `AnimalsDesktop-macos-amd64.zip` for Intel Macs on macOS 12+
- `SHA256SUMS.txt` for package verification

## What's Changed

- Added nineteen selectable animals: striped chipmunk, leopard gecko, blue White's tree frog, budgerigar, cockatiel, Java sparrow, parrotlet, lovebird, ragdoll, Scottish Fold, French Bulldog, Maine Coon, domestic shorthair calico, British Shorthair, Toy Poodle, Munchkin, Roborovski hamster, Russian smoke white guinea pig, and quokka.
- Promoted all nineteen into the Windows and macOS runtime selection lists and settings coverage.
- Updated GitHub Pages to show the current thirty-five-animal roster and the remaining planned silhouettes.
- Documented upcoming lanes on Pages: the remaining GitHub Pages candidates, a white lionhead-pattern rabbit, and a special low-motion shoebill.
- Repaired pre-release visual issues for the leopard gecko middle-turn scale and domestic shorthair contact baseline.

## Preview Scope

This remains a preview release. Each newly promoted animal has one accepted 62-frame source set and is imported into the runtime sprite sheets for behavior and settings QA. The full-content gate still expects expanded motion coverage before a stable/final release.

Extra props such as trees, seeds, food, and other foraging items are still out of scope for this preview.
