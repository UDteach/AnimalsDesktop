# AnimalsDesktop v0.2.13

Release date: 2026-07-03

## Changes

- Fixes Windows animal selection drift caused by older settings files that stored only numeric animal indexes.
- Adds stable `variantID` and `selectedCoatIDs` fields to newly saved Windows settings so future animal additions do not remap saved choices.
- Repairs the known legacy case where an older saved shoebill selection could appear as a cat after the v0.2.12 roster update.
- Groups the Windows animal picker by animal type in both the settings UI and tray animal menu.

## Runtime Scope

- Public selectable animals: 57.
- This is a hotfix release; the animal roster is unchanged from v0.2.12.
- Existing numeric settings continue to load, and new saves include both numeric indexes and stable IDs.
