# AnimalsDesktop v0.2.10

This is a preview release for 56 selectable animals on Windows and macOS.

## Changes

- Adds `chinchilla_beige` and `chinchilla_ebony` as color-tone variants derived from the accepted `chinchilla_standard_gray` motion source while preserving frame geometry and alpha.
- Replaces the black-eyed white chipmunk runtime slot with `true_albino_chipmunk`; the legacy black-eyed source remains only as source evidence.
- Refreshes `fancy_rat_blue_hooded` so the hood and spine read as a cooler gray-blue while preserving the white body, pink details, frame geometry, and alpha.
- Updates the public Pages preview, current animal icons, JP/EN page text, and release checks for the 56-animal roster.

## Known Preview Scope

The public runtime uses accepted `set00` motion sources duplicated into runtime sets. This remains a preview exception before the final full 10-set motion gate.
