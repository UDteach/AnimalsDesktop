# AnimalsDesktop v0.2.14

Release date: 2026-07-03

## Changes

- Extends the animal selection drift fix to both Windows and macOS settings.
- Saves animal selections by stable runtime variant IDs on Windows and macOS while keeping the older numeric index fields for compatibility.
- Groups Windows and macOS animal selection menus by broad animal type, such as rabbits, chinchillas, hamsters, birds, cats, dogs, and reptiles/amphibians.
- Recovers named shoebill settings such as `ハシビロコウ` / `shoebill` while leaving ambiguous legacy numeric indices unchanged because old settings do not store the source app version.
- Keeps the public selectable roster at 57 animals with no new runtime assets in this hotfix.

## Runtime Scope

- Public selectable animals: 57.
- New settings format: Windows settings version 3, macOS settings version 4.
- Future saves use stable IDs so roster order changes do not change the selected animal.
