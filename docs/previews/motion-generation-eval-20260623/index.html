Run 05 G: angle blueprint wording review-only

Generator Verdict

current choice
  • Final accepted 現時点は ImageGen を優先。Nano Banana Pro は hard frame の比較候補に追加するが、最終 accepted は1枚ずつ `prepareframe`、light/dark review、`auditframes -artifact-warnings` を通す。
  • Flow usage Flow / Nano Banana 2 は walk / scurry / turn のような連続動作の案出しに限定。まず 16セル、成功後に 31セル。62セルは探索扱い。
  • Next batch size 16セルで guide ink / pinhole が残ったため、次は8フレームの歩行ループだけを試す。可視グリッドは使わず、style anchor only の不可視2x4配置か、8枚独立出力を優先。
  • Do not reuse `transparent pinholes` が出た既存 frame は style anchor から除外。生成済みの frame-00 anchor grid pack は archive 済み。
  • Stop rule Flow / Nano Banana が guide ink、pinhole、face fail、source-family drift を出したら、同じ prompt/anchor/guide で再試行しない。
